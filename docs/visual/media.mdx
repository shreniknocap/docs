Automatically or manually add images, videos, and GIFs to your project

<div id="ios-tab">
    <Tabs>
        <Tab title="iOS">
            **Automatically add media with Magic Edit**

            1. Tap **Media**
            2. Tap **Magic Edit**
            3. Choose from the list of options to have media automatically added

            **Manually add or edit media**

            1. Tap **Media**
            2. Tap the **+** icon
            3. Import a picture or video from your camera roll, GIF, or stock photo

            **Edit Media**

            - Location - Use one finger to drag to a new location
            - Duration - Use one finger to drag the start or end of the media
            - Edit - Tap **Edit** to adjust the position or transition
            - Delete - Tap **Delete** or **Delete All**
        </Tab>

        <Tab title="Desktop">
